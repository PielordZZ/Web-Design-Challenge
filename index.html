Project information

This project is intended to analyze how various weather conditions are affected by the locations proximity to the equator. To provide analysis data from aproximetely 500 cities was collected from OpenWeather API

This project shows that of various weather phenominon only temperature has a coorilation with latitude. The other factors of cloudiness, wind speed, and humidity have no strong coorilation to the locations proximity to the equator.
